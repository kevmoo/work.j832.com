I've been jumping back and forth between two world for the last few years.
  1. World .NET - C#, WPF, Silverlight, WCF, Visual Studio, .NET Reflector, etc
  2. World Open Source Web - Ruby on Rails, Javascript, Chrome, TextMate, Sublime, Bash, etc.
I've talked to a lot of people about my two heads on this topic.

Coding in .NET (at least on a Microsoft platform) is painful once you've dealt with the wonder of the open source world. I'm never stuck thinking I'm going to be left with a bad issue. I can always get to the source of my problem--litterally--and fix things. It's wonderful.

But the upside of .NET is mostyl with the typeness of it.

I can click and GoTo definition. I can 'find all references'. I can open up Reflector and analyze.

This is amazing for my productivity.

I can understand the code.
